# Changes

This folder stores **changelogs produced by scaffold-managed updates**, especially migrations applied via Concierge (Migrator module).

## What belongs here

- Vitaixmen migration reports:
  - `workspace-migration-YYYY-MM-DD.md`

## What does not belong here

- Ongoing hygiene/AIX maintenance logs → use `docs/logs/`
- Project-specific operational artifacts → use `docs/logs/projects/<project-slug>/`
